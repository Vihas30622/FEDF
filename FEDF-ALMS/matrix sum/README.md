# Matrix Sum

Simple React app to calculate the sum of two matrices.

Files created:
- `src/MatrixSumCalculator.jsx` — main component with state and handlers
- `src/App.jsx` — app wrapper
- `src/index.js` — entry
- `src/Styles.css` — basic styles
- `public/index.html`
- `package.json`

Run (PowerShell):
